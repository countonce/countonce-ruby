# CountOnce-ruby

Wrapper for the CountOnce API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'countonce-ruby'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install countonce-ruby
